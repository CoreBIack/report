This static web page represents the Virtua shopping cart project. It is packaged in a Docker container using Nginx and deployed automatically after the CI checks pass on the main branch.

Functions covered by the test suite

  • add_item()
  • remove_item()
  • apply_discount()
  • get_total()
  • clear()
  • get_item_count()
CI Runs linting and the full Python unit test suite before merge.
Containerization Builds a Docker image using nginx:alpine and the static/ folder.
CD Deploys to Render automatically after a successful push to main.
